# Two-Tier-Web-Application (Flask + MySQL) with CI/CD using jenkins

In this project, I try to simulate a real-world two-tier application deployment using Docker and CI/CD practices.

The Goal of this project is not only to run web application, but also to undersatand Devops workflows work, 
from containerization -> secure credential management -> to automated deployment on a cloud server

Technologies used:

- Flask (Python) - Web Application
- MySQL - Database
- Docker & Docker Compose
- Jenkins (CI/CD Automation)
- AWS EC2 (Deployment Server)
